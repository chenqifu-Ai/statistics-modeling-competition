# 📊 数据收集清单

## 🎯 数据需求总览

### 核心数据维度
1. **医疗机构数据** - 医院数量、规模、资源配置
2. **医疗服务数据** - 诊疗人次、费用、效率指标  
3. **医保支出数据** - 总额、分类、增长趋势
4. **人口结构数据** - 年龄分布、老龄化程度
5. **社会经济数据** - 区域发展水平相关指标

## 🛠️ 数据管理规范

### 文件命名规范
```markdown
# 时间序列数据
YYYY_数据类别_变量名.csv
示例: 2024_hospital_count.csv

# 区域数据  
YYYY_区域_数据类别.csv
示例: 2024_Liaoning_health_expenditure.csv

# 分类数据
YYYY_分类_数据类别.csv
示例: 2024_TCM_service_volume.csv
```
